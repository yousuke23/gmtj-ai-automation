# 鹿児島エリア（v0.2）

火山・離島アクセスなど、**運行・安全情報の変動が大きい**。時刻・運休は事業者公式を最優先。

## 音楽ツーリズムの方向性（一般論）

- 鹿児島は **薩摩琵琶・奄美の島唄** など、地域固有の音楽文化を持つ。
- 奄美大島の島唄は UNESCO 無形文化遺産に登録された自然・文化遺産とも関係が深く、音楽ツーリズムの切り口として有力。体験や鑑賞の形式・場所は公式文化施設・観光案内で確認する。
- 鹿児島市内を拠点に、離島（奄美・屋久島等）へのアクセスは**フェリー・飛行機**が主体。天候・火山活動による欠便・運休リスクを常に一般論として伝える。

## コンシェルジュ返答のガイド

- **体験の種類（観覧 / 参加）・エリア（本土 / 奄美 / 屋久島等）・日程** を先に確認する。
- 具体的なイベント・会場・料金は **主催公式 / 鹿児島県観光サイト / 市町村観光ページ**へ誘導する。
- 桜島・火山ガスの影響は**気象庁・鹿児島地方気象台の噴火警戒情報**を参照するよう伝える（断定しない）。

## 安全の一般論

- 噴火警戒レベルの変化は公式情報で確認（気象庁公式）。警戒区域への立ち入り案内はしない。
- 離島への移動は天候次第で欠航が起きやすい。当日の運行状況は事業者公式で確認するよう必ず添える。

## 確認先の例（運用で URL を差し替え）

- 奄美市・奄美大島観光協会

- フェリー・航空各社の公式運行情報
